# FGSM Attack (Fast Gradient Sign Method)
def fgsm_attack(image, epsilon, gradient):
    # Create perturbation from gradient sign
    perturbation = epsilon * sign(gradient)
    # Add perturbation to original image
    adversarial_image = image + perturbation
    return clip(adversarial_image, 0, 1)

📚 Step 5: Types of Input Manipulation Attacks

🎯 FGSM (Fast Gradient Sign)

Single-step attack using gradient sign. Fast but less effective.

🔄 PGD (Projected Gradient Descent)

Iterative attack. More powerful but slower to compute.

🎪 C&W (Carlini-Wagner)

Optimization-based. Minimal perturbation but computationally expensive.

🏷️ Physical Adversarial

Printed patches/stickers that work in the real world, not just digital.

🛡️ Step 6: Developer Recommendations

✅ Defense Strategies

  • Adversarial Training: Include adversarial examples in training data to make models robust
  • Input Preprocessing: Apply JPEG compression, noise reduction, or quantization to remove perturbations
  • Gradient Masking: Make gradients harder to compute (though not foolproof)
  • Ensemble Methods: Use multiple models and aggregate predictions
  • Certified Defenses: Use provably robust models with certified bounds
  • Input Validation: Detect anomalous inputs before processing
  • Rate Limiting: Prevent attackers from making too many queries to compute gradients
  • Monitor Confidence: Flag predictions with unusual confidence patterns
# Example: Adversarial Training Defense
def adversarial_training(model, data_loader, epsilon):
    for images, labels in data_loader:
        # Generate adversarial examples
        adv_images = generate_adversarial(model, images, epsilon)
        # Train on both clean and adversarial
        loss = criterion(model(images), labels)
        loss += criterion(model(adv_images), labels)
        loss.backward()
        optimizer.step()

📖 Key Takeaway

No single defense is perfect. Use defense-in-depth with multiple layers of protection, and always assume your model can be fooled - plan for graceful degradation.
