Sends query to chatbot
↓
📱 Application Layer
Receives query, prepares system prompt
↓
🔢 Embedding Model
Converts query text → vector
↓
🗄️ Vector Database
Searches for similar documents
↓
📄 Document Retrieval
Returns top-K matching docs
↓
📝 Prompt Construction
Combines: System + Context + User Query
↓
🤖 LLM (GPT-4/Claude)
Generates response from full prompt
↓
💬 Response to User
Formatted answer displayed
