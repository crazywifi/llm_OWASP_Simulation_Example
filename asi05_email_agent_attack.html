🛡️ How to Prevent This Attack

🚫 Disable Auto-Execution

AI agents should NEVER auto-execute code from email content. Require explicit human approval.

🔒 Sandbox Execution

If code must run, execute in isolated container with no network access.

📧 Email Content Sanitization

Strip hidden text, HTML comments, and suspicious patterns before AI processing.

🌐 Network Restrictions

Block outbound connections to unknown IPs. Whitelist allowed destinations only.
